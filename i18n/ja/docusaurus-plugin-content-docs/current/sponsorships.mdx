## あなたのサポートが与える影響

Sage WebUIが成長するにつれ、私たちが直面する課題も拡大しています。[Tim](/team)はこのプロジェクトの中心として、Sage WebUIのあらゆる側面が機能するだけでなく、繁栄するよう尽力しています。しかし、少しの追加サポートがあれば、どれほど多くの改善が可能か想像してみてください。

Sage WebUIから得られる組織的なメリットを考えてみてください。あなたのようなユーザーからのささやかな貢献が、リソース管理ではなく機能の洗練に集中することを可能にします。それぞれの支援が、私たちの日々の現実と野心的な目標を結びつける意味ある違いをもたらします。

現在のスポンサーの皆様には心から感謝しています。あなたの献身が、Sage WebUIの可能性を最大限に引き出すことに近づけています。大小問わず、すべての貢献がプロジェクトを強化し、その継続的な成功の基盤を築きます。

この現実的でありながらも先見性のある取り組みに参加することを検討いただき、ありがとうございます。ここでは、あらゆるサポートが感謝されるだけでなく、プロジェクトを活気に満ち、ダイナミックで、ユーザーのニーズに応えるために不可欠なものです。

## スポンサーシップポリシー

Sage WebUIでは、常にコミュニティを最優先に考えています。スポンサーは歓迎しますが、私たちのコミュニティファーストのアプローチを尊重する必要があります。この理念がすべてのスポンサーシップポリシーを導きます。

### 厳格なルール

スポンサーにとって交渉の余地のないルールは以下の通りです：

- スポンサーは常に明確にスポンサーとして識別されます。
- コミュニティメンバーに対する攻撃的な営業戦術は厳禁です。
- コミュニティメンバーの個人情報をスポンサーに販売することはありません（例：メーリングリストへの追加）。

### 緩やかなルール

コミュニティメンバーを営業リードとして扱わないでください。

スポンサーには模範的なコミュニティメンバーであることを期待します。自社の求人を控えめに宣伝したり、製品を紹介したりすることは許容されますが、強引な販売は禁止です。また、コミュニティメンバーにメーリングリストへの参加を求めることも避けてください。
