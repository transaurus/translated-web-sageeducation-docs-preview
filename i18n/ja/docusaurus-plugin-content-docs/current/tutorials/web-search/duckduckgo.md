## DuckDuckGo API

### セットアップ

Sage Open WebUIの組み込みウェブ検索でDuckDuckGo APIを使用するために特別なセットアップは必要ありません！DuckDuckGoはSage Open WebUIでそのまま動作します。

:::note
ウェブ検索がレート制限される可能性があります。
:::
